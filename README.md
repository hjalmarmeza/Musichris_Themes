# 💎 MusiChris Themes | Universal Forge v1.1

**MusiChris Themes** es el motor de forja ministerial definitivo para la producción automatizada de contenido cristiano de alto impacto (YouTube Shorts / Vertical Videos). Este sistema transforma noticias del mundo real, enlaces y comentarios en reflexiones espirituales profundas a través del "Corazón de Dios".

## 🚀 Características Premium
- **Motor de Sabiduría (IA):** Impulsado por Cerebras Llama 3.1 para una forja instantánea de guiones ministeriales.
- **Visuales WOW:** Fondo dinámico de video (`Fondo_theme.mp4`) y logo animado con transparencia profesional.
- **Diseño Sacrosanta:** Layout centralizado que garantiza legibilidad y estética premium.
- **Audio Atmosférico:** Sincronización automática con la melodía oficial *Alas del Alba*.
- **Transiciones Cinematográficas:** Efectos de fundido (Fade) para una fluidez visual superior.

## 🛠️ Requisitos de Despliegue (Secrets)
Si planeas desplegar este motor en GitHub Actions o un servidor en la nube, debes configurar los siguientes **GitHub Secrets**:

1.  `CEREBRAS_API_KEY`: Clave necesaria para el motor de IA que forja los scripts.
2.  `DEEPSEEK_API_KEY`: (Opcional) Clave de respaldo para procesamiento lingüístico avanzado.

## 📈 SEO & Algoritmo (YouTube Shorts)
